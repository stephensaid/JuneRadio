# June Radio

##### The project includes
* Physical build of the body made from wood with PVC front panel
* PCB design for the front panel
* Parts list

##### Features
* Internet radio
* DAB Radio
* FM Radio (possibly)
* MP3 player
* Favorite stations
* Weather station
* Time and alarm functions
